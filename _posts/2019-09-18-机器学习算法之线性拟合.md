### 题目 
```
实现一个特征预处理类，将输入的特征归一化到0~1之间，
```
### 一种解法

```
def fit_linear_function(points):
    M = len(points)
    x_bar=np.mean(points[:,0])
    sum_yx= 0
    sum_x2=0
    sum_delta =0
    for i in range(M):
        x=points[i,0]
        y=points[i,1]
        sum_yx += y*(x-x_bar)
        sum_x2 += x**2
    k = sum_yx/(sum_x2-M*(x_bar**2))
    for i in range(M):
        x=points[i,0]
        y=points[i,1] 
        sum_delta += (y-k*x)
    b = sum_delta / M
    return k, b
```
